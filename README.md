# BMI Calculator

A simple and user-friendly BMI (Body Mass Index) calculator built with Python.

## Features
- Calculates BMI
- Shows weight category (Underweight, Normal, Overweight, Obese)
- Calculates Ideal Weight for Male and Female
- Clean and formatted output

## How to Run

1. Make sure Python is installed
2. Download `bmi_calculator.py`
3. Run this command:

```bash
python bmi_calculator.py
